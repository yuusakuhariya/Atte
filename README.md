## 機能一覧
・メールアドレス、パスワードでのログイン
<br/>
・名前、メールアドレス、パスワードの新規登録
・出勤記録、退勤記録
・休憩開始、休憩終了の複数回記録
・1日の出勤時間計算、1日の休憩時間計算
・指定日の各ユーザーの勤務記録の表示
